# SH-IMG2TAR

## Help

img2tar

    Usage: img2tar [-o TARFILE][-n PARTITION] DISK
    
    From an image "DISK" get partition "PARTITION" and create
    a tar file.
    
    Requires: mount/umount/losetup/tar/fdisk/tar
